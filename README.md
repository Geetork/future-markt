# Web page React/TS/MUI

## Tasks
1. HTML/CSS/SCSS:

- Create an HTML file and build a web page using HTML5 and CSS/SCSS.
- Ensure semantic HTML markup and utilize modern CSS properties.
- Utilize SCSS for managing styles, including separating styles into modules and using variables and mixins for reusability.
- Ensure responsiveness of the page and its appealing appearance on various devices and browsers.
- Guarantee accessibility and keyboard navigability of functional elements.
2. JavaScript:

- Add interactivity to the web page using JavaScript.
- Implement the display of additional content on the page when clicking on consultation buttons.
- Perform simple form validation, checking for the completion of all mandatory fields before submission.
- Create basic animations for elements on the page, such as smooth fading in or out.
- Store data in LocalStorage for persistence.
3. JS Framework:

- Implement the project using the React or Vue3 framework.
- Fetch the current exchange rate of the currency "GBP" from the endpoint https://www.cbr-xml-daily.ru/daily_json.js and round it to the nearest integer.
- Calculate the sum of the digit values of the current date and utilize it in the project. For example, if today is 01.01.2000, the sum would be 0 + 1 + 0 + 1 + 2 + 0 + 0 + 0 = 4.

## Getting started
1. Clone the repository.
2. Run npm install to install dependencies.
3. Execute npm run start to start the development server.
